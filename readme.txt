== Description ==

The *Shockingly Big IE6 Warning* is a plugin that shows a warning message alerting the user why it is bad to use IE6, the security risk and the bad compatibility of Web Standards.

You can choose from 3 types of warning:

* TOP, a discreet top bar
* FULL, a fullscreen warning
* CRASH, the mean option

**Recent Changes**

* Fixed the CRASH mode
* Some links erros
* Almost everything recoded
* Now uses wordpress jQuery not an external .js
* Browsers download urls now editable
* Test mode, to preview the warning in any browser

See the new [Shockingly Big IE6 Warning homepage](http://www.incerteza.org/blog/projetos/shockingly-big-ie6-warning/ "Shockingly Big IE6 Warning") for further information.

Thanks for everyones suggestions!

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install it thought the admin interface
2. Choose the type of warning and save
3. DONE!
